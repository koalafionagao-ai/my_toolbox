# my_toolbox
## Notion-2-Github v6.0 big full version on 2026/03/22
 6 Modules × 21 features：
📦 模块一：核心文件与解析引擎
1. 多格式支持：支持拖拽上传 Notion 导出的 .zip 或单篇 .html / .htm 文件。
2. ZIP 深度遍历：如果是 ZIP 包，能自动忽略 __MACOSX 等系统隐藏文件，精准找到 HTML 正文并提取所有附属图片。
3. Notion 冗余代码清洗：自动剥离 Notion 导出的原生页眉、侧边栏、原生 Meta 标签等不需要的代码，仅保留 <article> 核心正文内容。
4. 元数据（Meta）智能提取：自动解析 Notion 原生表格中的 Created（创建时间）、Updated（更新时间）和 Tags（标签），提取数据后自动销毁原表格。
🖼️ 模块二：图片画质引擎与 CDN 分发
5. 本地 WebP 极速压缩：利用浏览器 Canvas 引擎，将提取到的所有图片（PNG/JPG等）在本地纯前端压缩转换为 .webp 格式（最大边长限制 1920px，质量 80%）。
6. 图片资源打包下载：提供 images.zip 打包下载按钮，供你推送到 GitHub。
7. 图片路径自动替换：将 HTML 中乱码的原生图片路径，全部精准替换为你配置的 GitHub 路径。
8. 智能双轨 CDN 链路：
  - 注入 data-native（你的 GitHub 原生路径）和 data-cdn（JsDelivr 中国区加速路径）。
  - 生成的 HTML 底部自带一段 JS 探针脚本：判断访客如果在东八区（中国），则自动激活 CDN 链接，否则使用原生链接。
9. LQIP 极致防白屏（毛玻璃懒加载）：为每张图片生成极其微小的 20px Base64 占位图。在真实图片加载完成前，显示带有 blur(10px) 的高斯模糊底图，加载后丝滑变清晰。
💅 模块三：高级页面排版与 UI 增强 (生成的博客页面)
10. 全局现代化 CSS 注入：覆盖 Notion 原生丑陋的字体和间距，注入了一套极其优雅的系统级字体（San Francisco / Apple-system）和响应式排版。
11. 自动构建博客页头（Meta 区域）：在 H1 标题下方自动生成精美的页头，包含：
  - 提取出的标签 (Tags) 阵列。
  - 带有邮箱 Icon 的作者徽章（点击一键复制邮箱 koala.fiona.gao@gmail.com，并有“已复制”动画提示）。
  - 首页 (Home) 链接。
  - 日历 Icon + 创建时间；时钟 Icon + 更新时间（如果存在）。
  - 语言切换功能（点击自动在当前 URL 中互换 /zh/ 和 /en/ 路径）。
12. 版权页脚（Footer）注入：页面最底部自动注入版权年份、作者链接和严谨的中英文双语免责声明。
13. 极客级代码块增强：
  - 将代码块放入独立的 .code-wrapper 中，实现横向滚动隔离（防穿模）。
  - 左上角显示语言标签（如 Python, JS）。
  - 右上角提供沉浸式纯净 Outline 风格的“一键复制代码”图标。
  - 自动引入 Prism.js 实现代码高亮。
🧭 模块四：TOC 悬浮目录大纲 (Tocbot 引擎)
14. 锚点自动修补：如果 Notion 导出的标题缺少 id，工具会自动用正则为 H1-H3 补全 id，确保跳转有效。
15. PC 端优雅横线随动目录：右侧居中的隐藏式目录。未悬浮时是极简横线（带绿色阅读进度条），悬浮时完整展开，剔除了所有原生丑陋竖线，多行长标题完美换行展示。
🚀 模块五：SEO、分享与预览
16. 微信朋友圈首图 Hack：在 <body> 最顶部偷偷植入宽度为 0 的高清封面图，专治微信爬虫抓不到图导致朋友圈分享空白的顽疾。
17. 社交分享卡片 (OG Tags)：自动生成 Twitter Card 和 Facebook OG 标签，并将你在工具“第 4 步”中选定的图片（支持自动筛选小于 500k 的轻量图或随机默认壁纸）设为封面。
18. SEO 摘要智能注入：自动提取正文首段作为默认 Description，支持在工具“第 5 步”手动修改并一键更新到代码中。
19. 所见即所得的极简预览：
  - 工具内点击“预览样式”打开新标签页。
  - 逻辑隔离：自动剔除 data-cdn 和模糊滤镜，将图片临时替换为“绿框虚线 SVG 占位图”（带提示文字），确保本地预览极其清晰稳定，不受死链干扰。
🛠️ 模块六：工具自身基础能力
20. 中英文双语界面：右上角一键切换工具自身的语言 UI（完全响应国际化）。
21. 压缩前后图片画廊：可随时点击单张图片预览压缩前后的体积对比。
